ACO 420: Large-scale Data Management (Spring 2015)

Project 2: Large-Scale Messaging System with HBase

Due Tuesday, March 17th 2015, before class starts.

This is a group project (two students per group). Weight: 15% of class grade.

Implement a messaging system in Java:

  • create the table in HBase. It will have: emailAddress, Date, Time, Title, Body
  • send message button = write a single record to the database for a single user - To.
  • refresh button = fill combo boxes. Big one with message Body, small one with emailAddress. Return logged in user's messages.
  • Pre-load the system with a dataset that contains 5,000 messages from 500 users (each user sent 10 messages to other users).

Deliverables (Send by email to the instructor in a single Zip file):

  1. The source code of the system (Java files).
  2. A single-page document (PDF) explaining:
    1. Why HBase is a good data store for this application
    2. What is the structure of the created HBase table(s)
    i. List the column families and their columns ii. Explain the structure of the row keys

    Points will be deducted from your grade for failing to follow the above directions.
